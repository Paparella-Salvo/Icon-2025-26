# Type Weather
Progetto sviluppato per il corso di Ingegneria della Conoscenza (a.a. 2025/2026)  
Università degli Studi di Bari “Aldo Moro”

## Descrizione del progetto
Type Weather è un sistema ibrido che combina modelli di apprendimento supervisionato e Reti Bayesiane per la classificazione delle condizioni meteorologiche.
L’obiettivo è analizzare come diverse tecniche di rappresentazione della conoscenza e di apprendimento influenzino la capacità predittiva del modello.

Il progetto include:

- preprocessamento e discretizzazione del dataset meteorologico

- addestramento di modelli supervisionati (KNN, Decision Tree, Random Forest, SVM)

- costruzione e valutazione di tre Reti Bayesiane:

  - struttura appresa con K2

  - struttura appresa con BIC

  - struttura Expert definita manualmente

- confronto tra modelli in termini di accuratezza e complessità

- analisi della Knowledge Base e della struttura delle BN
